---
title: Seeded randomness
description: Random walks that replay identically
---

Scenes get Effect's `Random` service seeded out of the box, so "random"
choices are reproducible: this walker visits the same six points on every
run. Replay the scene — the path never changes. Pass a different `seed` in
the runner settings to get a different (equally reproducible) walk.
