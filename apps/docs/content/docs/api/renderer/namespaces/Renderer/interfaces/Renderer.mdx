# Interface: Renderer

Defined in: [Renderer.ts:132](https://github.com/julia-script/effect-motion/blob/main/packages/renderer/src/Renderer.ts#L132)

A live browser renderer.

## Remarks

Mostly data — the API is the sibling functions ([syncFrame](../functions/syncFrame.mdx),
[resolveResources](../functions/resolveResources.mdx), [render](../functions/render.mdx), [prewarm](../functions/prewarm.mdx)). `sync.stats`
is useful for diagnostics: it reports how many objects are retained and
how long the last sync took.
