# Interface: Font\<ID\>

Defined in: [packages/motion/src/Font.ts:16](https://github.com/julia-script/effect-motion/blob/main/packages/motion/src/Font.ts#L16)

A font reference as entity data stores it: `{ _tag, id }`. The id is the
family name text paints register and select by. Obtain one by yielding a
[Font](../variables/Font.mdx) constant inside a scene — that is what puts the matching
`FontLoader<ID>` into the scene's requirements.
