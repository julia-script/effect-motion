# Interface: Wait()

Defined in: [packages/motion/src/Motion.ts:706](https://github.com/julia-script/effect-motion/blob/main/packages/motion/src/Motion.ts#L706)

The dual nature of [wait](../functions/wait.mdx): an Effect you can yield, and a function
you can drop into a `pipe` chain.

## Remarks

Being callable is what lets a hold sit BETWEEN two animators in the same
chain — it wraps the preceding step, waits after it completes, and passes
that step's value (the instance) through untouched, so the chain keeps
flowing.
