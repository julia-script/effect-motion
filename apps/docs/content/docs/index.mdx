title: Introduction
description: Declarative, deterministic animations built on Effect
---

**effect-motion** is an animation library where scenes are written as
[Effect](https://effect.website) generator programs: instantiate entities,
tween or spring their properties, and compose motions sequentially or in
parallel. Scenes are **deterministic** — seeded randomness and a frame-locked
clock make every run byte-identical — and **finite**, which is what lets the
player below pause, scrub, and replay them like a video.
