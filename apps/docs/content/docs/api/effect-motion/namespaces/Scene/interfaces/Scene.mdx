# Interface: Scene\<E, R\>

Defined in: [packages/motion/src/Scene.ts:76](https://github.com/julia-script/effect-motion/blob/main/packages/motion/src/Scene.ts#L76)

A scene: an animation body plus the composition it plays in.

## Remarks

Inert on its own — building one runs no animation and produces no frames.
It is a description that [run](../functions/run.mdx), [stream](../functions/stream.mdx), or [play](../functions/play.mdx)
later executes, which is what lets the same scene be replayed, nested, or
rendered at different settings without change.

## Type Parameters

| Type Parameter | Default type | Description |
| ------ | ------ | ------ |
| `E` | `never` | How the scene can fail. |
| `R` | `never` | What it needs to run (fonts, images, and other resources). |
