Like [dollyTo](./dollyTo.mdx), but starting from an explicit distance.

## Remarks

For a push-in that begins further out than the camera currently sits —
the move starts at `from` regardless of where the camera was left.

## Param

**from**

Starting distance from the point of interest.

## Param

**to**

Target distance.

## Param

**duration**

How long, in scene time.

## Param

**timing**

An easing name or function.

## Returns

The camera, so animators chain.
