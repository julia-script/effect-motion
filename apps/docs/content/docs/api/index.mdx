## Where to start

Most scenes only touch a handful of modules:

| Module | For |
| --- | --- |
| [`Scene`](/docs/api/effect-motion/namespaces/Scene) | Declaring a scene, creating entities, composing animations |
| [`Motion`](/docs/api/effect-motion/namespaces/Motion) | Animating over a duration, with easing |
| [`Physics`](/docs/api/effect-motion/namespaces/Physics) | Springs — motion whose length emerges from a simulation |
| [`Entity`](/docs/api/effect-motion/namespaces/Entity) | The shapes, and `vec3` for positions |
| [`Color`](/docs/api/effect-motion/namespaces/Color) | Building and adjusting colors |
| [`Timing`](/docs/api/effect-motion/namespaces/Timing) | The easing curves |
| [`Camera`](/docs/api/effect-motion/namespaces/Camera) | Aiming, orbiting, and dollying the viewpoint |

## Regenerating

The reference is generated from the TSDoc comments in each package's source
and committed, so the docs site builds without running TypeScript:

```bash
pnpm docs:api
```

Re-run it after changing a doc-comment or a public signature.
