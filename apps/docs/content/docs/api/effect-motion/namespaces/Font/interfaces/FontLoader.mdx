Defined in: [packages/motion/src/Font.ts:34](https://github.com/julia-script/effect-motion/blob/main/packages/motion/src/Font.ts#L34)

A loaded font, provided as a context service: bytes are already in memory
by the time any consumer reads this (loads run eagerly at layer
construction — see [layer](../functions/layer.mdx)). `format` is optional; consumers sniff
magic bytes when absent.
