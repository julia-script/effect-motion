---
title: Easing race
description: Same tween, three different timing functions
---

Three circles travel the same distance in the same two seconds — the only
difference is the timing function. `Phaser.all` runs the tweens in parallel;
the third argument of `Motion.tweenTo` names any built-in easing (or takes a
custom `(t: number) => number`).
